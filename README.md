# Minecraft 1D ⛏️

**Minecraft 1D** — это минималистичный демейк культовой игры, перенесенный в **одномерное пространство**. Весь мир представляет собой одну бесконечную (или ограниченную) линию блоков, где вы можете перемещаться только вперед и назад, добывать ресурсы, строить и выживать.

---

## 🚀 Особенности проекта

* **1D Мир:** Все блоки и сущности расположены на одной линии.
* **Генерация мира:** Процедурная генерация биомов (пещеры, леса, пустыни) в одном измерении.
* **Сбор и крафт:** Добыча базовых ресурсов (земля, камень, дерево, руда) и создание инструментов.
* **Механика выживания:** Полоска здоровья, голод и динамическая смена дня и ночи.
* **Враги:** Одномерные мобы, атакующие игрока при сближении.

---

## 🎮 Управление

* `A` / `D` (или `←` / `→`) — Перемещение влево и вправо.
* `Левая кнопка мыши` (ЛКМ) — Разрушить блок перед собой.
* `Правая кнопка мыши` (ПКМ) — Поставить выбранный блок.
* `1` - `9` — Выбор активного слота в хотбаре.
* `E` — Открыть инвентарь / меню крафта.
* `ESC` — Пауза.

## 🛠️ Установка и запуск

### Требования
Для запуска игры вам понадобится установленный [Python 3.x](https://python.org) и менеджер пакетов `pip`.

### Инструкция

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com
   ```

2. Перейдите в папку с проектом:
   ```bash
   cd minecraft-1d
   ```

3. Установите необходимые зависимости:
   ```bash
   pip install -r requirements.txt
   ```

4. Запустите игру:
   ```bash
   python main.py
   ```

---

## 📂 Структура проекта

* `main.py` — Главный файл для запуска игры и инициализации игрового цикла.
* `world.py` — Логика генерации одномерного мира и хранения блоков.
* `player.py` — Класс игрока, физика движения, инвентарь и параметры здоровья.
* `sprites/` / `assets/` — Папка с текстурами блоков, спрайтами мобов и звуками.
* `requirements.txt` — Список библиотек, необходимых для работы (например, `pygame`).
